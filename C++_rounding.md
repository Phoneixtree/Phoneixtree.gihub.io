# 1.直接赋值给整数变量。如：
>int i = 2.5;   
>或 i = (int) 2.5;  
>这种方法采用的是舍去小数部分

# 2.整数除法运算符
>“/”本身就有取整功能(int/int)  
>整数除法对正数的取整是舍去小数部分

# 3.使用floor函数
>floor(x)返回的是小于或等于x的最大整数,如:  
>floor(2.5) = 2  
>floor(-2.5) = -3

# 4.使用ceil函数
>使用ceil函数。ceil(x)返回的是大于x的最小整数,如:  
>ceil(2.5) = 3  
>ceil(-2.5) = -2
